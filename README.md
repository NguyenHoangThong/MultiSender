# MultiSender Contract

1. Install and setup
   - Copy env/.env.example to env/.env
   - Change private key using to deploy
   - Run command:
    ```shell
    yarn install
    ```
2. Testing
    ```shell
    yarn test
    ```
3. Deploy
    ```shell
    yarn run deploy-bsctestnet
    ```
4. Verify contract
    ```shell
   yarn run verify-bsctestnet 
   ```
